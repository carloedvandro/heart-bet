## Instalação e Configuração

### Pré-requisitos
- Node.js 18+
- npm ou yarn
- Conta Supabase

### Configuração do Frontend
```bash
# Instalar dependências
npm install

# Desenvolvimento
npm run dev

# Build
npm run build
```

### Variáveis de Ambiente Necessárias
```
SUPABASE_URL=sua_url
SUPABASE_ANON_KEY=sua_chave
```
